Социальная сеть реализованная с помощью spring boot + vue.js проект для портфолио.

Регистрация OAuth2 через google аккаунт.

Реализовано:
  сообщения;
  коментарии к сообщениям;
  предпросмотр ссылок на сайты по микроразметке Open Graph Protocol и Twitter Cards;
  подписки на пользователей с подтверждением, без подписки нельзя видеть сообщения других пользователей.
  
Планирую добавить:
  группы;
  смену фотографии пользователя;
  загрузку файлов.
